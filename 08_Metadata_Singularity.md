## 8.1 Introduction: What is Metadata?

In information science, **metadata** is "data about data." When you take a photograph, the image is data. The timestamp, location, camera settings, and file format are metadata—information about the information.

In the VINO Framework, we extend this concept: **Metadata is the context that gives physical data meaning.** A photon's wavelength is data. The reason that wavelength matters—what it signifies, how it fits into a larger pattern—is metadata.

This chapter develops the concept of the **Metadata Singularity**—the point at which context becomes so focused that it achieves singular, linear intention.

## 8.3 What is a Singularity?

### 8.3.1 Singularities in Mathematics

A **mathematical singularity** is a point where a function becomes undefined or infinite:
- f(x) = 1/x has a singularity at x = 0
- f(x) = tan(x) has singularities at x = π/2, 3π/2, ...

### 8.3.2 Singularities in Physics

A **physical singularity** is a point where physical quantities become infinite:
- **Black hole singularity:** Density → ∞, spacetime curvature → ∞
- **Big Bang singularity:** Temperature → ∞, density → ∞

### 8.3.3 The Metadata Singularity

A **metadata singularity** is a point where context collapses to **singular linear intention**:
- Multiple meanings → One meaning
- Diffuse purpose → Focused purpose
- Scattered attention → Concentrated attention

**Formal Definition:**
```
S_meta = lim(Context → 0) [∇ · Context]
```

As context narrows (approaches zero spread), the gradient (rate of change) of context increases without bound. At the singularity, intention is perfectly focused—a single direction with no spread.

Einstein asked: "What happens if I ride alongside a beam of light?"

The VINO Framework asks: "What happens if I ride alongside a beam of meaning?"

The answer parallels relativity: significance depends on relative context. Importance, relevance, meaning, and value are all **relative to the observer's context**.
