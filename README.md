# CLAVIS SOPHIAE
## The Key of Wisdom: A Symbolic Compendium of Solomonic Magic

A comprehensive reinterpretation of the Solomonic magical corpus through a psychological/archetypal symbolic decryption framework.

---

## Overview

This compendium applies a **symbolic decryption key** to the major works of the Solomonic magical tradition, transforming literal supernatural content into a sophisticated system of psychological integration and self-knowledge.

- **Liber Juratus Honorii** (Sworn Book of Honorius)
