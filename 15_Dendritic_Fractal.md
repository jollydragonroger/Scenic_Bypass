## 15.2 What is a Fractal?

### 15.2.1 Definition

A **fractal** is a mathematical object with self-similarity across scales. Zoom in on a fractal, and you see the same (or statistically similar) pattern repeating.

**Key Properties:**
- **Self-similarity:** Parts resemble the whole
- **Fractional dimension:** Not integer (e.g., dimension 1.26, not 1 or 2)
- **Infinite detail:** Zooming reveals ever-finer structure
- **Simple rules, complex output:** Fractals arise from simple iterative processes

### 15.2.2 The Mandelbrot Set

The most famous fractal, defined by:
```
z_{n+1} = z_n² + c
```

Starting with z₀ = 0, iterate. If the sequence remains bounded, c is in the Mandelbrot set.

The boundary of this set is infinitely complex, with the same patterns appearing at all scales.

### 15.2.3 Natural Fractals

Fractals appear throughout nature:
- **Coastlines:** Jagged at every scale
- **Mountains:** Similar ridges from meters to kilometers
- **Clouds:** Self-similar turbulent structures
- **Trees:** Branching patterns repeat from twig to trunk
- **Rivers:** Delta patterns match tributary patterns
- **Blood vessels:** Branching from aorta to capillaries
- **Neurons:** Dendritic trees with self-similar branching
- **Lightning:** Branching discharge patterns
- **Galaxies:** Filamentary cosmic web

## 15.3 What is a Dendrite?

### 15.3.1 Biological Definition

A **dendrite** (from Greek δένδρον, "tree") is the branching extension of a neuron that receives signals from other neurons.

**Properties:**
- Branching structure with multiple levels
- Information flows inward (toward cell body)
- Each branch can receive from multiple sources
- The pattern is self-similar at different scales

### 15.3.2 The Dendritic Pattern

The dendritic pattern appears in many contexts:

- River systems (water flow optimization)
- Lightning (electrical discharge paths)
- Frost patterns (crystal growth)
- Lung bronchi (gas exchange optimization)
- Root systems (nutrient absorption)

**Common Feature:** All these systems optimize flow or information transfer through branching.

## 15.5 Mathematical Description

### 15.5.1 Fractal Dimension

The fractal (Hausdorff) dimension D measures "how much" space a fractal fills:
- D = 1: A line (fills 1D completely)
- D = 2: A plane (fills 2D completely)
- D = 1.26: The Koch snowflake boundary (between line and plane)
- D = 2.58: A rough surface (between plane and volume)

### 15.5.2 The Reality Dimension

**Hypothesis:** The dendritic fractal of reality has dimension:

Where φ = (1 + √5)/2 ≈ 1.618 is the golden ratio.

### 15.5.3 The Branching Ratio

In dendritic systems, the **branching ratio** r describes how branches divide:
```
r = (diameter of parent) / (sum of diameters of children)
```

For optimal flow (Murray's law):
```
r³_parent = r³_child1 + r³_child2 + ...
```

**For Reality:**
```
Scale_parent³ = Σ Scale_children³
```

The "scale" of a phenomenon cubes to equal the sum of the cubed scales of its sub-phenomena.
